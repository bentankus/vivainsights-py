# Version 0.2.0

The new version adds a number of organizational network analysis (ONA) functions to the library: 

- Network visualization and analysis:
    - `network_g2g()`
    - `network_p2p()`
    - `network_summary()`
    - `create_sankey()`
- Sample / simulate datasets: 
    - `p2p_data_sim()`
    - `load_g2g_data()`
    - `load_p2p_data()`
